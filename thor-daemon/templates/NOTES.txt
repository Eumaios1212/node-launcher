- Get the THORChain Node IP by running these commands:

  export NODE_IP=$(kubectl exec -it --namespace {{ .Release.Namespace }} deploy/{{ include "thor-daemon.name" . }} -- curl -s http://whatismyip.akamai.com)
  echo $NODE_IP

- Visit RPC endpoint of the THORChain Node:

  Mainnet:
  curl http://$NODE_IP:27147

  Testnet / Mocknet:
  curl http://$NODE_IP:26567
